# SpaceExplorer
Collector, processor, and provider of space data from various sources.

## Data Sources

This project may ingest data from NASA, NOAA, JPL, CelesTrak, Minor Planet Center,
and other public APIs. The project license applies to this repository's source
code, not necessarily to third-party data, trademarks, logos, or external API
content.

Users should review and comply with the terms, attribution requirements, and
usage guidelines of each upstream data provider.
